# Praktikum Linked List (Python)

## Deskripsi
Program ini merupakan implementasi sederhana dari **Single Linked List** menggunakan bahasa Python.  
Linked List adalah struktur data yang terdiri dari node-node yang saling terhubung.

## Fitur
- Tambah data di awal
- Tambah data di akhir
- Hapus data
- Menampilkan seluruh data

## Struktur Program
- `Node` → untuk menyimpan data dan pointer ke node berikutnya
- `LinkedList` → untuk mengelola operasi linked list

## Cara Menjalankan

1. Pastikan Python sudah terinstall
2. Jalankan file dengan perintah:

```bash
python nama_file.py
